## Stack

- Pure HTML + CSS — no framework, no build step
- Custom `<x-dc>` web component system (`support.js`) for case study pages
- Assets self-hosted in `assets/` and `crops/`
- Deployed via **GitHub Pages** (branch: `main`, root `/`)

## Updating the site

```bash
# Make changes locally in D:\Prasanna\Portfolio, then:
git add <files>
git commit -m "your message"
git push origin main
# GitHub Pages rebuilds automatically in ~1 minute
```
